## key, value 를 이용해 value 호출하기  

```swift
var phoneNumbers = ["Jenny": "867-5309", "Mateo": "510-7752", "Mike": "330-8004",
    "Alicia": "489-4608", "Daniel": "455-2626", "Josie": "769-3339"]
```
이렇게 phoneNumbers dictionary가 있을 때, Alicia의 번호를 가져오려면 어떻게 해야할까?  

```swift
phoneNumbers["Alicia"]
```

이렇게 해주면된당!
